# Park Cleanup Evidence Checklist

## Before Cleanup
- [ ] Date and time documented
- [ ] Weather conditions noted
- [ ] GPS coordinates / park name confirmed
- [ ] Wide-angle "overview" photo of park area
- [ ] Close-up photos of litter/debris (3-5 minimum)
- [ ] Photos from consistent reference points (note landmarks)
- [ ] Optional: Short video walkthrough

## During Cleanup
- [ ] Number of volunteers documented
- [ ] Start time noted
- [ ] Trash bags used counted
- [ ] Notable items collected (e.g., bottles, food wrappers, cigarette butts)
- [ ] Optional: Progress photos

## After Cleanup
- [ ] End time noted
- [ ] Same reference point photos as "before"
- [ ] Wide-angle "overview" from same positions
- [ ] Close-ups of cleaned areas
- [ ] Photo of collected trash bags
- [ ] Bag count and estimated weight
- [ ] Optional: Short video walkthrough

## Documentation
- [ ] Upload all photos to evidence/ folder in this repo
- [ ] Create a cleanup report (use templates/cleanup-report-template.md)
- [ ] Update candidate park status in README.md
